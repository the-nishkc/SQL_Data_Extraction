📊 Supermarket Sales SQL & Data Extraction

📌 Project Overview

This project demonstrates SQL and Python integration using the Supermarket Sales dataset. The objective is to perform data extraction, aggregation, and business analysis using SQLite and Python.

🎯 Objectives

- Connect Python with SQLite
- Execute SQL queries
- Extract business insights
- Perform data aggregation and analysis
- Demonstrate SQL fundamentals and advanced queries

## 🛠 Technologies Used

- Python
- SQLite
- Pandas
- VS Code
